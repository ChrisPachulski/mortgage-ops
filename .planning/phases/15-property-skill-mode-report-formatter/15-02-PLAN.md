|----------|-------------|
| AnalysisReport (Pydantic) → render() | AnalysisReport is frozen+strict from Phase 14; trust = full. render() consumes via field access; no parsing. |
| orchestrator_argv (list[str]) → footer | argv values come from sys.argv[1:] of the orchestrator (Plan 15-03); the orchestrator validates --listing/--household/--profile/--output-dir before calling render(). Trust = sanitized-CLI-args. |
| render() output (str) → caller (orchestrator) | Pure string; no executable content. Caller decides where to write. |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-15-04 | Tampering | render() embedding user-controllable PropertyListing.address in markdown | accept | Markdown is not executed; GitHub/VS Code markdown renderers escape raw HTML conservatively. Personal-use scope. Residual risk documented in RESEARCH L954. If reports are ever served to a browser, add an HTML-escape pass. |
